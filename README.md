# 🧮 Calculator Web App

My very first web development project built while learning Full Stack Development.

This project is a simple calculator application created using HTML, CSS, and JavaScript. It performs basic arithmetic operations through a clean and modern user interface.

---

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Decimal number support
- Clear display button
- Delete last character button
- Error handling for invalid expressions
- Responsive calculator layout using CSS Grid

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## ⚙️ How It Works

### HTML
Creates the calculator layout including:
- Display screen
- Number buttons
- Operator buttons
- Clear, Delete, and Equals buttons

### CSS
Provides:
- Grid layout
- Dark theme styling
- Rounded buttons
- Hover and active effects

### JavaScript
Handles:
- Updating the display
- Performing calculations
- Clearing input
- Deleting characters
- Error handling
